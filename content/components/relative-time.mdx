### Past date-time

For past date-times the relative time outputs different formats depending on the time difference.

#### Within a month

It shows relative date phrases in the format of `X seconds/minutes/days ago`. If the time difference is less than 1 minute, it shows `just now`.

#### After a month

It shows the date in the format `on D MMM` or `on D MMM, YYYY` depending on the current year.

### Future date-time

It strictly shows relative date phrases in the format of `in X seconds/minutes/hours/days/months/years`. This will never switch to displaying the date, even after a month has passed.

### Micro

The `micro` variant shows a smaller version of the relative time. It is useful for displaying the relative time in a more compact space or in a mobile environment. It shows the relative time in the format of `Xs/h/d`, `D MMM` after a month or `D MMM, YYYY` after a year.

## Usage

A relative time is intended to be used within a sentence, following the action that it's relative to.

A relative time can be used in isolatation when the content and the sourronding elements make it clear what the time is relative to.

A relative time is used as plain text in most of the cases. However, it can be used as a link to point to a section within a view that contains relevant information.

## Accessibility

A relative time is treated like a [time](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/time) element when it shows a local numerical date, and it contains an [abbreviation](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/abbr) element when it shows a relative date phrase.
